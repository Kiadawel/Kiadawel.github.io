Usage

This site was hand-coded by Kimi Weldon as a design and coding project for CIT230: Web Frontend Development at BYU-Idaho. The contents therein are intended entirely for educational purposes. Articles, blog posts, event data and town information are created expressly for this purpose and should not be used as factual data sources.

Weather Data

All weather data including current conditions, forecasting, temperatures, wind speed, etc. provided by the API at OpenWeatherMap.org and used with gratitude. Data is as accurate as the API can provide, specific to the stated locations. (Note: This uses a free ATT-ID account. Please do not over-refresh the town pages and freeze the API.)

Images

Photographic images for Tri-Town Weather Watch, including article/blog headers, hero images, homepage town-info cards and gallery photos are stock photos (not actual local photography) sourced from Pexels.com
